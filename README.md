# Shop Builder 🛍️

ShopBuilder est une plateforme permettant de connecter des clients avec des développeurs via un système de matching basé sur plusieurs critères (compétences, projets, avis).

## Technologies utilisées
- React.js
- CSS

## Installation

git clone https://github.com/VOTRE_USERNAME/shop_builder.git
cd shop_builder
npm install
npm start
